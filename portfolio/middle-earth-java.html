Introduction

The Middle Earth Java application was designed to allow students who use the web application to pull data from the web application and to allow them to make edits or create their own articles to push to the website from a convenient Java application.

The application was developed after the API was implemented in the Laravel web application. An Ant build script also helps users to run the application.

Initial Ideas

I had never developed a Java application before, yet alone one that has a graphical user interface, so this was a definite learning experience for me. I also learned how to create an Ant build script which would clean and compile my code.

Since I had already built the API that it was going to use, I knew that the hardest part would be coming up with a layout for the GUI that would be intuitive. There were four main pages in the program that one could use: Latest Articles, Update Profile, Create Article, and Search for articles.

Final Concept

While it is fully functional, I would like to go back and enhance the UI for users. Still, I enjoyed developing an application that utilized an API that I created. It also used the API to authenticate users so that they could use the application; unauthenticated users were not able to use it. It was a fun learning experience for me!
